b) Coding value function iteration¶

Before I can use my code, I need to rewrite my reward function in term of capital and labour only:

$$e^{q}i=k^{\prime}-(1-\delta)k$$


$$i=\frac{k^{\prime}-(1-\delta)k}{e^{q}}$$ Finding consumption: $$c=k^{\alpha}h^{1-\alpha}-i$$ $$c=k^{\alpha}h^{1-\alpha}-\frac{k^{\prime}-(1-\delta)k}{e^{q}}$$ Substituting into the reward function: $$R=\theta log(c)+(1-\theta)log(1-h)$$ $$R=\theta log(k^{\alpha}h^{1-\alpha}-\frac{k^{\prime}-(1-\delta)k}{e^{q}})+(1-\theta)log(1-h)$$

Checking the first order condition to find labour function: $$\frac{\partial R}{\partial h}=\theta (1-\alpha) \frac{k^{\alpha} h^{-\alpha}}{k^{\alpha}h^{1-\alpha}-\frac{k^{\prime}-(1-\delta)k}{e^{q}}}-\frac{1-\theta}{1-h}=0$$
The condition would be used to calculate the labour function.

From Problem set 4 my AR for log q coefficient is 0.817924 and the standard deviation of innovation is 0.01069. I am writing my code in such a way that one can easily run it with different values

Problems with algorithm in slides and my modifications:¶

Many of the grid points are simply infeasible as amount of investment can not be supported by the amount of labour available (c_max in the code). In labour_calculating function I fixed this problem, by calculating the minimum ammount of labour required for the state to be feasible. If the sate is infeasible NaN was returned

Second problem was that my bysection algorithm did not find solutions for many feasible cases, because initial values were of the same sign. To work this problem out, I written my own optimisation algorithm (it is in the labour function):

1) Try bisection with inital point [c_max+0.01,0.99]

2) If bysection does not work, try newton method with initial point c_max

3) If nothing works return NaN
I feel that this method capture the possible choice space quite well.

Part c)¶

The policy function iteration algorithm requires solving $N_{k}xN_{s}$ equation system for each value of the value function V. To formulate the problem I am using the following transformations: $$V^{t}(k_i,z_s)=R^{t}(k_i,g_{k}^t(k_i,z_s),z_s)+\beta \sum_{s^{\prime}=1}^{N_z}\pi(z_s,z_{s^\prime})V^t(g_{k}^t(k_i,z_s),z_{s^\prime})$$ I am rewriting this into matrix form:
$$I vec(V^{t})=R-\beta M vec(V^{t})$$, where R is exactly $R^{t}(k_i,g_{k}^t(k_i,z_s),z_s)$ evaluated at the policy function.
M is matrix: $$ M= \left[ {\begin{array}{ccccccc} 0 & 0 & 0 & \pi(z_s,z_{s^\prime})& 0& 0\\ 0 & 0 & \pi(z_s,z_{s^\prime})& 0 & 0& 0 \\ ... & ... & ... & ...& ...& ...\\ 0 & 0 & 0& 0 & 0& \pi(z_s,z_{s^\prime}) \\ \end{array} } \right] $$, where position of $\pi(z_s,z_{s^\prime})$ in each row depends on the policy function.

The following simplifies to : $(I+\beta M) vec(V^{t})=R$ , which can be solved by Gaussian elimination. After that, I can undo vec operator on the resulting array to get $V^{t}$

The initial policy function I will choose is the one generated by the first iteration of VFI.
